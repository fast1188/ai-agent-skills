# Code Review Prompt Template

请审查以下代码,从以下维度评估:

1. **正确性**:逻辑错误、边界条件、异常处理
2. **可读性**:命名、注释、代码风格
3. **性能**:时间复杂度、空间复杂度、潜在瓶颈
4. **安全性**:输入验证、SQL 注入、XSS、密钥泄露
5. **可维护性**:模块化、复用性、扩展性
6. **测试覆盖**:是否有测试,测试质量

对每个问题:
- 指出具体位置(行号 / 函数名)
- 解释问题影响
- 给出修复建议

格式:
```
问题:[类型]
位置: [file:line]
影响: [严重程度]
建议: [修复方案]
```
